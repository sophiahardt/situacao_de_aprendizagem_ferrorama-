# Sistema Web para Gerenciamento de Operações Ferroviárias em Tempo Real

## Descrição do Projeto

A proposta é criar um sistema web simples e eficiente para gerenciar e monitorar operações ferroviárias em tempo real, integrando sensores, trens e relatórios analíticos em uma única plataforma. Nesse sistema, os administradores ou usuários já cadastrados poderão fazer login e acessar um painel com todos os sensores, trens e relatórios gerados. Apenas o administrador tem a possibilidade de atualizar ou excluir sensores, além de acompanhar informações como velocidade, localização e status dos trens de forma clara e visual. A plataforma terá identificação de falhas e tomada de decisões baseada em dados, garantindo maior controle operacional e segurança.
