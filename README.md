# Sistema Web para Gerenciamento de Operações Ferroviárias em Tempo Real

## Descrição do Projeto

A proposta é criar um sistema web simples e eficiente para gerenciar e monitorar operações ferroviárias em tempo real, integrando sensores, trens e relatórios analíticos em uma única plataforma. Nesse sistema, os administradores ou usuários já cadastrados poderão fazer login e acessar um painel com todos os sensores, trens e relatórios gerados. Apenas o administrador tem a possibilidade de atualizar ou excluir sensores, além de acompanhar informações como velocidade, localização e status dos trens de forma clara e visual. A plataforma terá identificação de falhas e tomada de decisões baseada em dados, garantindo maior controle operacional e segurança.

## Identidade Visual Do Projeto

A identidade visual do Sistema Ferroviário foi desenvolvida de forma colaborativa pela equipe, com o objetivo de transmitir modernidade e eficiência aos usuários. Buscamos adotar um design prático, simples e intuitivo, priorizando a experiência do usuário com o sistema. Os campos de login e cadastro apresentam fundo branco com texto em preto, garantindo a clareza na escrita e fácil leitura. Na tela de login utilizamos a imagem de um trem representar visualmente o sistema e reforçar sua temática.Além disso,aplicamos botões que combinam textos em negrito e ícones, facilitando a navegação e tornando a interface mais simples. A paleta de cores foi definida da seguinte forma: o branco se destaca sobre o azul, proporcionando melhor legibilidade. As cores azul e branco transmitem segurança e confiabilidade, enquanto o vermelho é utilizado para alertas e ações críticas. Já o verde indica elementos ativos ou em funcionamento, contribuindo para uma comunicação visual clara e eficiente.
